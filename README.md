# SpacetimeDB — Elixir Client

An Elixir client for [SpacetimeDB](https://spacetimedb.com) v2 — WebSocket
subscriptions, reducer and procedure calls, BSATN binary protocol, live ETS
table mirrors, and a code generator that builds schema modules from a live
database.

## Installation

```elixir
def deps do
  [
    {:spacetimedb_ex, "~> 0.3"}
  ]
end
```

## Features

- **Dual protocol** — BSATN binary (`v2.bsatn.spacetimedb`, default, 3–5× smaller) or JSON text
- **Full message coverage** — all v2 client→server and server→client message types
- **Reducers + Procedures** — call reducers and v2 procedure functions
- **`SpacetimeDB.BSATN.Schema`** — macro DSL: declare a table schema, get binary encode/decode free
- **`SpacetimeDB.Table`** — ETS-backed live table mirror; reads are pure ETS, no GenServer roundtrip
- **`mix spacetimedb.gen`** — connects to a live SpacetimeDB instance and generates all schema modules automatically
- **URI-based connection** — `uri: "https://maincloud.spacetimedb.com"` like the official SDKs
- **Auto-reconnect** with exponential backoff
- Works as a supervised child in any OTP application

## Quick start

```elixir
{:ok, conn} = SpacetimeDB.start_link(
  host: "localhost",
  database: "my_module",
  handler: %{
    on_connected: fn conn_info, _ -> IO.puts("Connected as #{conn_info.identity}") end,
    on_transaction_update: fn update, _ ->
      Enum.each(update.query_sets, fn qs ->
        Enum.each(qs.tables, fn t ->
          IO.puts("#{t.table_name}: +#{length(t.inserts)} -#{length(t.deletes)}")
        end)
      end)
    end
  }
)

SpacetimeDB.subscribe(conn, ["SELECT * FROM Player"])
{:ok, _req_id} = SpacetimeDB.call_reducer(conn, "CreatePlayer", ["Alice"])
```

### Connecting to SpacetimeDB Cloud (Maincloud)

```elixir
{:ok, conn} = SpacetimeDB.start_link(
  uri: "https://maincloud.spacetimedb.com",
  database: "my_module",
  handler: MyApp.Handler
)
```

## Code generation

Point the generator at a running SpacetimeDB instance and it writes one
`bsatn_schema` module per table:

```bash
# Local server
mix spacetimedb.gen \
  --host localhost \
  --database my-module \
  --namespace MyApp.SpacetimeDB \
  --out lib/my_app/spacetimedb/

# Maincloud
mix spacetimedb.gen \
  --uri https://maincloud.spacetimedb.com \
  --database my-module \
  --namespace MyApp.SpacetimeDB \
  --out lib/my_app/spacetimedb/
```

Generated file — `lib/my_app/spacetimedb/player.ex`:

```elixir
# Generated by SpacetimeDB.CodeGen — do not edit manually.
defmodule MyApp.SpacetimeDB.Player do
  @moduledoc "Live mirror of the `Player` SpacetimeDB table. Auto-generated."

  use SpacetimeDB.BSATN.Schema

  bsatn_schema do
    field :id,     :u32
    field :name,   :string
    field :health, :u32
    field :alive,  :bool
  end

  def primary_key, do: :id
  def table_name,  do: "Player"
end
```

Re-run any time the SpacetimeDB module schema changes.

## Live table mirrors

`SpacetimeDB.Table` keeps a named ETS table in sync with the server.
Reads are direct ETS lookups — no process call, callable from anywhere.

```elixir
# supervision tree
children = [
  {SpacetimeDB.Table,
   name: :players,
   schema: MyApp.SpacetimeDB.Player,
   table_name: "Player",
   primary_key: :id},

  {SpacetimeDB,
   uri: "https://maincloud.spacetimedb.com",
   database: "my-module",
   handler: MyApp.Handler}
]

# handler — apply updates to the mirror
defmodule MyApp.Handler do
  @behaviour SpacetimeDB.Handler

  @impl true
  def on_subscribe_applied(sub, _),
    do: SpacetimeDB.Table.apply_update(:players, sub)

  @impl true
  def on_transaction_update(update, _),
    do: SpacetimeDB.Table.apply_update(:players, update)

  @impl true
  def on_reducer_result(result, _),
    do: SpacetimeDB.Table.apply_update(:players, result)
end

# query from anywhere — pure ETS speed
SpacetimeDB.Table.all(:players)
SpacetimeDB.Table.get(:players, 1)
SpacetimeDB.Table.filter(:players, fn p -> p.alive end)
SpacetimeDB.Table.count(:players)
```

## BSATN protocol (default)

The connection defaults to `protocol: :bsatn` — binary WebSocket frames that
are 3–5× smaller than JSON. Row data arrives as raw BSATN binaries and is
decoded by your schema module:

```elixir
# Rows in TableUpdate.inserts are raw binaries in BSATN mode
{:ok, player, ""} = MyApp.SpacetimeDB.Player.decode(row_binary)
```

Switch to JSON with `protocol: :json` if you prefer plain decoded maps.

### Encoding reducer args in BSATN mode

```elixir
args =
  SpacetimeDB.BSATN.encode_string("Alice") <>
  SpacetimeDB.BSATN.encode_u32(100)

SpacetimeDB.call_reducer(conn, "CreatePlayer", args)
```

### Manual schema definition

If you prefer to define schemas by hand instead of generating them:

```elixir
defmodule MyApp.Player do
  use SpacetimeDB.BSATN.Schema

  bsatn_schema do
    field :id,     :u32
    field :name,   :string
    field :health, :u32
  end
end
```

Supported field types: `:bool`, `:u8`, `:u16`, `:u32`, `:u64`, `:u128`,
`:i8`, `:i16`, `:i32`, `:i64`, `:i128`, `:f32`, `:f64`, `:string`, `:bytes`,
`{:array, T}`, `{:option, T}`.

## Handler behaviour

All callbacks are optional. Implement only what you need:

```elixir
defmodule MyApp.Handler do
  @behaviour SpacetimeDB.Handler

  @impl true
  def on_connected(conn_info, _arg) do
    MyApp.Auth.store_token(conn_info.token)
  end

  @impl true
  def on_transaction_update(update, _arg) do
    Enum.each(update.query_sets, fn qs ->
      Enum.each(qs.tables, fn t ->
        MyApp.Cache.apply_diff(t.table_name, t.inserts, t.deletes)
      end)
    end)
  end

  @impl true
  def on_reducer_result(result, _arg) do
    IO.inspect(result.outcome, label: "reducer result")
  end

  @impl true
  def on_disconnect(reason, _arg) do
    MyApp.Metrics.record_disconnect(reason)
  end
end
```

## Connection options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `:uri` | `String.t()` | — | Full URI (e.g. `"https://maincloud.spacetimedb.com"`). Extracts host/port/tls. |
| `:host` | `String.t()` | required (unless `:uri`) | SpacetimeDB host |
| `:port` | `non_neg_integer()` | `3000` | Port (443 when TLS detected from URI) |
| `:tls` | `boolean()` | `false` | Use TLS (`wss://`) |
| `:database` | `String.t()` | required | Database name or identity hex |
| `:token` | `String.t() \| nil` | `nil` | Auth token (persisted on reconnect) |
| `:protocol` | `:bsatn \| :json` | `:bsatn` | Wire protocol |
| `:handler` | `module \| {module, term} \| map` | required | Callback module or function map |
| `:reconnect` | `boolean()` | `true` | Auto-reconnect on disconnect |
| `:reconnect_delay_ms` | `non_neg_integer()` | `500` | Initial reconnect backoff delay |
| `:max_reconnect_delay_ms` | `non_neg_integer()` | `30_000` | Maximum reconnect backoff delay |
| `:name` | `GenServer.name()` | — | Registered process name |

## Architecture

```
mix spacetimedb.gen
  └── SpacetimeDB.CodeGen        fetch schema → generate bsatn_schema modules

SpacetimeDB (public API)
  └── SpacetimeDB.Connection     GenServer — WebSocket lifecycle, auto-reconnect
        ├── SpacetimeDB.Protocol          JSON encode/decode (v2)
        ├── SpacetimeDB.Protocol.BSATN    binary encode/decode (v2)
        ├── SpacetimeDB.BSATN             binary codec primitives
        ├── SpacetimeDB.BSATN.Schema      macro DSL for table schemas
        ├── SpacetimeDB.Types             typed structs for all messages
        └── SpacetimeDB.Handler           optional callback behaviour

SpacetimeDB.Table                ETS-backed live table mirror
```
